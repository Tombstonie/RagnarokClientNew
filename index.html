Часто задаваемые вопросы

Почему при запуске вылезает ошибка?
Установите Java 17+. Это необходимо для корректной работы клиента.
Почему игра вылетает?
Выделите больше ОЗУ в лаунчере. Рекомендуется минимум 4 ГБ.
Почему мало FPS?
Используйте Java 17 вместо Java 21 для лучшей стабильности.

Руководство по сложным функциям

Auto Creeper Farm

Автоматическая охота на криперов

Как использовать

1. Встаньте на 1 угол вашей фермы и пропишите .creeper pos1
2. Встаньте на 2 угол вашей фермы и пропишите .creeper pos2
3. Включите функцию Auto Creeper Farm

Cocoa Farm

Фарм какао бобов

Как использовать

1. Встаньте на 1 угол вашей фермы и пропишите .cf pos1
2. Встаньте на 2 угол вашей фермы и пропишите .cf pos2
3. Включите функцию Cocoa Farm

Auto Event Loot

Авто лут ивентов фантайм

Как использовать

— В модуле ElytraHelper включите Автопрыжок
— Назначьте бинд Swap Elytra на клавишу H
— В модуле ChestStealer назначьте бинд на клавишу K

Необходимые предметы:
— Алмазка з5
— 2 стака фейерверков
— 16 золотых яблок
— Золотая морковь
— Тотем бессмертия
— Ботинки с зачарованием «Невесомость IV»
