Часто задаваемые вопросы

Почему при запуске вылезает ошибка?
Установите Java 17+. Это необходимо для корректной работы всех модулей клиента.
Почему моя игра вылетает?
Выделите больше ОЗУ в настройках лаунчера. Рекомендуемое значение — от 4 ГБ.
Почему у меня мало FPS?
Поменяйте Java 21 на Java 17. Стабильность клиента лучше всего проявляется именно на этой версии.

Руководство по сложным функциям

Auto Creeper Farm
Автоматически строит платформу для фарма криперов.

Как использовать:
1. Встаньте на нужную высоту.
2. Включите модуль в меню клиента.
3. Убедитесь, что в инвентаре есть полублоки и люки.
Air Stuck
Позволяет зафиксировать игрока в воздухе.

Как использовать:
1. Подпрыгните или слетите с блока.
2. Активируйте функцию горячей клавишей.
3. Для отключения нажмите повторно.
Swing Animation
Меняет анимацию удара руки.

Как использовать:
1. Откройте настройки клиента.
2. Выберите стиль анимации.
3. Примените изменения.
